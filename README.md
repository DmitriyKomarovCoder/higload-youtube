# Проектирование высоконагруженных приложений

## 1. Тема и целевая аудитория

### Тема "Проектирование сервиса по типу Youtube"

### Функционал MVP:
- загрузка видео
- просмотр видео
- авторизация
- оценка видео (лайки, дизлайки)
- подписка на канал
- комментарии к видео
- Поиск видео/канала
